# 🌍 Travelgram

Projeto desenvolvido durante a trilha de HTML e CSS da **[Rocketseat](https://rocketseat.com.br/)**, com o objetivo de praticar estruturação semântica com HTML e estilização com CSS puro.

## 📋 Sobre o projeto

O **Travelgram** é uma página de perfil inspirada em redes sociais de viagem. A interface apresenta:

- **Barra de navegação** com logo, links e avatar do usuário
- **Seção de perfil** com foto, bio e informações (localização, países visitados e fotos)
- **Galeria de fotos** em grid responsivo com efeito de zoom ao passar o mouse
- **Rodapé** com links de termos e privacidade

## 🚀 Tecnologias

- HTML5 semântico
- CSS3 (Flexbox, variáveis CSS, transições e filtros)
- Google Fonts — [Poppins](https://fonts.google.com/specimen/Poppins)

## 🗂️ Estrutura de pastas

```
Rocketseat-Travelgram/
├── index.html
├── style/
│   ├── reset.css       # Zeragem de estilos padrão do browser
│   └── main.css        # Estilos principais da aplicação
└── assets/
    ├── icons/          # SVGs de ícones (logo, navegação, perfil)
    └── images/         # Fotos do perfil e galeria
```

## 🎨 Design System

As cores e fontes são centralizadas em variáveis CSS (`:root`):

| Variável | Valor | Uso |
|---|---|---|
| `--brand-color` | `#EF5F4C` | Cor de destaque (laranja) |
| `--bg-profile` | `#FFFFFF` | Fundo do perfil |
| `--surface-color` | `#F5F5F5` | Fundo do header |
| `--text-primary` | `#313131` | Texto principal |
| `--text-secondary` | `#6C6C6C` | Texto secundário |
| `--font-main` | `Outfit` | Fonte principal |

## ⚙️ Como executar

Por ser um projeto estático (HTML + CSS puro), basta abrir o arquivo no navegador:

```bash
# Clone o repositório
git clone https://github.com/obrunofeitosa/Rocketseat-Travelgram.git

# Abra o arquivo index.html no seu navegador
```

Ou, se tiver o VS Code com a extensão **Live Server**, clique em `Go Live` na barra inferior.
